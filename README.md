Project Name: Green Morse

Minimum: A browser message is sent to a Raspberry Pi, which converts the text into Morse code flashes on an LED and matching buzzer beeps.

Wow: The system translates any typed message in real time into synchronized light and sound Morse code output, creating an interactive communication demo.

Demo story: A user types “HELLO” in the browser, clicks send, and instantly watches the LED blink and hears the buzzer beep the Morse code sequence live from the Pi.
